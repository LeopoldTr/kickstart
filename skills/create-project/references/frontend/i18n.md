# i18n Reference

## Library per Framework

| Framework | Library | Pattern |
|-----------|---------|---------|
| Next.js | `next-intl` | Colocated `.messages.json`, localePrefix: "as-needed" |
| React (Vite) | `react-i18next` | JSON files in `locales/{lang}/` |
| Vue | `vue-i18n` | JSON files in `locales/{lang}/` |

## Next.js (next-intl)

### Routing (`i18n/routing.ts`)
```typescript
export const routing = defineRouting({
  locales: ["en", "fr"],
  defaultLocale: "en",
  localePrefix: "as-needed"
});
```

### Request Config (`i18n/request.ts`)
Imports and merges colocated `.messages.json` files per namespace.

### Navigation (`i18n/navigation.ts`)
```typescript
export const { Link, redirect, usePathname, useRouter } = createNavigation(routing);
```

### Colocated Messages
Message files live next to their component: `header.messages.json`, `home.messages.json`.

## React (react-i18next)

```
src/locales/
├── en/
│   ├── common.json
│   └── dashboard.json
└── fr/
    ├── common.json
    └── dashboard.json
```

Setup in `lib/i18n.ts`, wrap app in `I18nextProvider`.

## Vue (vue-i18n)

Same file structure as React. Setup in `lib/i18n.ts`, register as Vue plugin in `main.ts`.
